📲 Installer le jeu comme une vraie appli

Voici comment installer l'appli "Un voyage jusqu'à moi" sur ton téléphone pour y jouer facilement à tout moment.

🤖 Sur Android (Chrome, Brave...)

1. Clique sur le lien ci-dessous pour ouvrir le jeu
2. Attends ou clique un peu, un message apparaîtra : « Ajouter à l'écran d'accueil »
3. Si rien ne s'affiche, clique sur les 3 points (menu) en haut à droite
4. Choisis « Ajouter à l'écran d'accueil », puis valide

🍎 Sur iPhone (Safari uniquement)

1. Ouvre Safari et entre le lien
2. Appuie en bas sur l'icône de partage (le carré avec une flèche vers le haut)
3. Choisis « Sur l'écran d'accueil »
4. Clique sur Ajouter

🚀 Lancer le jeu maintenant
